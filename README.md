# 球弈 · 数据解码每一注

> 2026 FIFA世界杯 竞彩分析平台 | v3.3

## 概述

球弈是面向中国体育彩票竞彩足球的智能分析工具，覆盖比赛数据、方案生成、比分预测、球队对战等功能。专为2026美加墨世界杯设计，支持全赛段（小组赛→决赛）。

**访问**: http://124.220.72.141:8899/

---

## 功能矩阵

| 标签 | 功能 | 数据源 | 说明 |
|:---|:---|:---|:---|
| 📊 比赛 | SPF/让球赔率 + 比分推荐 | sporttery.cn | 每场3×SPF + 3×RQ + 最佳比分 |
| 📋 方案 | 过关组合优化 | 后端算法 | 2-4串组合，EV排序，按预算分配 |
| 🏆 夺冠 | 夺冠赔率分析 | sporttery.cn | 各队夺冠概率排序 |
| 📈 复盘 | 历史战绩回看 | 本地数据 | 已结束比赛的真实比分 |
| ⚔️ 对战 | AI胜率预测 | 算法模型 | 机器学习预测+赔率对比 |
| 🎁 商城 | 积分兑换 | 本地 | 积分系统，兑换虚拟物品 |

### 比赛详情弹窗 (v3.3)
点击任意比赛卡片 → 弹出 SPF + RQ + 比分Top10 完整赔率表

```

│   │   ├── /api/plan           # 投注方案
```

### 技术栈
- **后端**: Python 3.8+ / FastAPI / uvicorn

- **数据**: sporttery.cn 竞彩官网
- **部署**: 腾讯云 Lighthouse Windows + TAT 自动化

## 前端约束 (ES5 铁律)

目标浏览器（含Hermes内置引擎）仅支持 ES5:

| ES6 | ES5 替代 |
|:---|:---|
| `const` / `let` | `var` |
| `async function` | `function` + `.then()` |
| `await` | `.then(回调)` |
| `() => {}` | `function(){}` |
| `` `模板${x}` `` | `'拼接'+x` |
| 解构 `[a,b]` | `var p=arr; a=p[0]` |

**关键**: 单处 ES6 语法错误 → 整个 `<script>` 块不执行。

## 部署流程

1. 修改本地代码
2. 生成 ES5 兼容版本 (无 const/let/=>/async)
3. gzip 压缩 + base64 编码
4. 通过腾讯云 TAT API 推送到 Windows 服务器
5. 重启 uvicorn 进程

```powershell
# 服务器启动命令
$env:PYTHONIOENCODING="utf-8"
Start-Process python -ArgumentList "-X","utf8","-m","uvicorn","main:app","--host","0.0.0.0","--port","8899"
```

## 数据格式

### SPF + 让球赔率
```
法国 VS 摩洛哥
  SPF[0]: 胜1.40 平3.85 负6.45
  RQ[-1]: 胜2.43 平3.13 负2.51
```
- 前3个 = 胜平负; 后3个 = 让球
- [0] = 无让球SPF, [-1] = 主队让1球

### 方案响应 (/api/plan)
```json
{
  "core": [
    {"picks":["法国-1胜","不让球胜"],"sp":3.6,"win_rate":20.6,"ev":0.74,"stake":100,"payout":360}
  ],
  "value": [],
  "total_stake": 500,
  "max_payout": 1080
}
```
